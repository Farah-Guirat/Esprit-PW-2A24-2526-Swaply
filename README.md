# Swaply

## Description
This project was developed as part of the Web Programming (PW) – 2nd Year Integrated Preparatory Program at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

It is a web application that allows users to exchange services.

## Features
- User authentication
- Profile management
- Posting offers and requests

## Tech Stack
- HTML
- CSS
- JavaScript
